- The popover menu is hidden by default (`visibility: hidden`) until opened.

              └── Multiple: MuiGrid-root undefined Mui-classesWidget-OFS-classList2 Mui-classesWidget-OFS-disabledBooking MuiGrid-container MuiGrid-justify-content-xs-space-between (each class row, one for each class)

-------------------
1. MuiGrid-root hapana-base
   - The main "block" class
   - Top-level container
   - ID: "classes"

2. Toastify
   - Inside main block
   - Doesn't seem to be visible
   - ID: "classes"

3. MuiContainer-root Mui-class-OFS-root_container MuiContainer-maxWidthLg
   - The main area
   - Inside the main block

4. MuiGrid-root
   - Inside the main area container

5. MuiGrid-root Mui-classListHeader-OFS-root MuiGrid-container
   - The heading banner
   - Contains calendar picker on the left and login on the right
   - Same HTML level as the classes widget
   - Contains:
     a. MuiGrid-root MuiGrid-item MuiGrid-grid-xs-12
        - Contains MuiPaper-root Mui-classListHeader-OFS-paper
        - Calendar picker section: MuiGrid-root Mui-classListHeader-OFS-borderRight
          * Button: MuiButton-root Mui-button-OFS-textButton Mui-button-OFS-textPrimary Mui-classListHeader-OFS-dateFilter
          * Contains: Calendar icon, date text (Mui-classListHeader-OFS-textGray64), dropdown icon
        - Login section: MuiGrid-root Mui-classListHeader-OFS-clientLogin
          * Button: MuiButton-root Mui-button-OFS-textButton Mui-button-OFS-textSecondary Mui-classListHeader-OFS-userLogin
          * Contains: User icon, "Login" text
          * Popover menu for logout

6. MuiGrid-root Mui-classesWidget-OFS-p0 MuiGrid-container
   - "A day" container
   - Same HTML level as the class list header
   - Contains:
     a. MuiGrid-root Mui-classesWidget-OFS-item2 Mui-classesWidget-OFS-listHead MuiGrid-item MuiGrid-grid-xs-12
        - Day header with title

     b. Multiple instances of: MuiGrid-root undefined Mui-classesWidget-OFS-classList2
        - Each class row, one for each class
        - May have: Mui-classesWidget-OFS-disabledBooking or Mui-classesWidget-OFS-pointer

          * Session name section: MuiGrid-root Mui-classesWidget-OFS-item2 Mui-classesWidget-OFS-sessionName (grid-xs-true)

  └── .Mui-calendar-OFS-picker
      └── .MuiPickersDesktopDateRangeCalendar-root

Date Button States (for CSS targeting):
----------------------------------------
Each date is a button with class .MuiPickersDay-root inside .MuiPickersDateRangeDay-root

1. START DATE (first selected date):
   - Container: .MuiPickersDateRangeDay-root.MuiPickersDateRangeDay-rangeIntervalDayHighlightStart
   - Button: .MuiPickersDay-root.Mui-selected
   - Preview wrapper: .MuiPickersDateRangeDay-rangeIntervalDayPreviewStart

2. END DATE (last selected date):
   - Container: .MuiPickersDateRangeDay-root.MuiPickersDateRangeDay-rangeIntervalDayHighlightEnd
   - Button: .MuiPickersDay-root.Mui-selected
   - Preview wrapper: .MuiPickersDateRangeDay-rangeIntervalDayPreviewEnd

3. IN-BETWEEN DATES (dates between start and end):
   - Container: .MuiPickersDateRangeDay-root.MuiPickersDateRangeDay-rangeIntervalDayHighlight
   - Button: .MuiPickersDay-root.MuiPickersDateRangeDay-dayInsideRangeInterval
   - Note: These show the range highlight background

4. OUTSIDE RANGE (not selected, outside the date range):
   - Button: .MuiPickersDay-root.MuiPickersDateRangeDay-dayOutsideRangeInterval
   - May also have: .MuiPickersDateRangeDay-notSelectedDate

5. TODAY (current date):
   - Button: .MuiPickersDay-root.MuiPickersDay-today

6. DISABLED DATES:
   - Button: .MuiPickersDay-root with Mui-disabled class or disabled attribute

7. NOT SELECTED (available dates not in range):
   - Button: .MuiPickersDay-root.MuiPickersDateRangeDay-notSelectedDate
