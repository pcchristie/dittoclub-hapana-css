Hapana Widget HTML Structure Reference
======================================

Main Structure (Simplified):
----------------------------
MuiGrid-root hapana-base (main block)
  ├── Toastify (not visible)
  └── MuiContainer-root Mui-class-OFS-root_container MuiContainer-maxWidthLg (main area)
      └── MuiGrid-root
          ├── MuiGrid-root Mui-classListHeader-OFS-root MuiGrid-container (heading banner: calendar picker left, login right)
          └── MuiGrid-root Mui-classesWidget-OFS-p0 MuiGrid-container (a day)
              ├── MuiGrid-root Mui-classesWidget-OFS-item2 Mui-classesWidget-OFS-listHead MuiGrid-item MuiGrid-grid-xs-12 (day header with title)
              └── Multiple: MuiGrid-root undefined Mui-classesWidget-OFS-classList2 Mui-classesWidget-OFS-disabledBooking MuiGrid-container MuiGrid-justify-content-xs-space-between (each class row, one for each class)

Detailed Breakdown:
-------------------
1. MuiGrid-root hapana-base
   - The main "block" class
   - Top-level container
   - ID: "classes"

2. Toastify
   - Inside main block
   - Doesn't seem to be visible
   - ID: "classes"

3. MuiContainer-root Mui-class-OFS-root_container MuiContainer-maxWidthLg
   - The main area
   - Inside the main block

4. MuiGrid-root
   - Inside the main area container

5. MuiGrid-root Mui-classListHeader-OFS-root MuiGrid-container
   - The heading banner
   - Contains calendar picker on the left and login on the right
   - Same HTML level as the classes widget
   - Contains:
     a. MuiGrid-root MuiGrid-item MuiGrid-grid-xs-12
        - Contains MuiPaper-root Mui-classListHeader-OFS-paper
        - Calendar picker section: MuiGrid-root Mui-classListHeader-OFS-borderRight
          * Button: MuiButton-root Mui-button-OFS-textButton Mui-button-OFS-textPrimary Mui-classListHeader-OFS-dateFilter
          * Contains: Calendar icon, date text (Mui-classListHeader-OFS-textGray64), dropdown icon
        - Login section: MuiGrid-root Mui-classListHeader-OFS-clientLogin
          * Button: MuiButton-root Mui-button-OFS-textButton Mui-button-OFS-textSecondary Mui-classListHeader-OFS-userLogin
          * Contains: User icon, "Login" text
          * Popover menu for logout

6. MuiGrid-root Mui-classesWidget-OFS-p0 MuiGrid-container
   - "A day" container
   - Same HTML level as the class list header
   - Contains:
     a. MuiGrid-root Mui-classesWidget-OFS-item2 Mui-classesWidget-OFS-listHead MuiGrid-item MuiGrid-grid-xs-12
        - Day header with title
        - Contains: h4.MuiTypography-root.Mui-classesWidget-OFS-h4 (e.g., "Monday, November 24")
     
     b. Multiple instances of: MuiGrid-root undefined Mui-classesWidget-OFS-classList2
        - Each class row, one for each class
        - May have: Mui-classesWidget-OFS-disabledBooking or Mui-classesWidget-OFS-pointer
        - Attributes: mb="4", session-id, session-date, booking-status, session-status, credit-required, casual-rate, id="classList-X-Y"
        - Contains 4 main sections:
          * Time section: MuiGrid-root Mui-classesWidget-OFS-item2 (grid-xs-3)
            - Contains: h4 (time), p.MuiTypography-body1 (duration)
          * Session name section: MuiGrid-root Mui-classesWidget-OFS-item2 Mui-classesWidget-OFS-sessionName (grid-xs-true)
            - Contains: h3.MuiTypography-h3 (class name), p.MuiTypography-body2 (room name)
          * Spots left section: MuiGrid-root Mui-classesWidget-OFS-item2 Mui-classesWidget-OFS-block4 (grid-xs-4)
            - Contains: h4.status (e.g., "10 spots left") or h4.error (e.g., "Closed")
          * Button section: MuiGrid-root Mui-classesWidget-OFS-item2 Mui-classesWidget-OFS-block5 (grid-xs-5)
            - Contains: button.MuiButton-root.Mui-button-OFS-button (Book button) or h4 (status text)

7. Load More Section
   - MuiTypography-root Mui-loadMore-OFS-loadMore
   - Contains button: MuiButton-root Mui-button-OFS-button Mui-button-OFS-outlinedSecondary

Key Classes for CSS Targeting:
-------------------------------
- Main container: .hapana-base or .MuiGrid-root.hapana-base
- Root container: .Mui-class-OFS-root_container
- Header banner: .Mui-classListHeader-OFS-root
- Calendar button: .Mui-classListHeader-OFS-dateFilter
- Login button: .Mui-classListHeader-OFS-userLogin
- Day container: .Mui-classesWidget-OFS-p0
- Day header: .Mui-classesWidget-OFS-listHead
- Class row: .Mui-classesWidget-OFS-classList2
- Time display: .Mui-classesWidget-OFS-classList2 h4 (first h4 in row)
- Session name: .Mui-classesWidget-OFS-h3
- Room name: .Mui-classesWidget-OFS-subTitle
- Spots left: .Mui-classesWidget-OFS-classList2 h4.status
- Book button: .Mui-classesWidget-OFS-classList2 button
- Load more: .Mui-loadMore-OFS-loadMore

Notes:
------
- The class list header and classes widget are at the same HTML level
- Each day section contains one day header and multiple class rows
- Class rows use the classList2 class and may have:
  * disabledBooking class when booking is disabled
  * pointer class when clickable
- The "X spots left" text is the second-to-last h4 in the classList2 container
- Status text can be either in h4.status (spots left) or h4.error (closed)
- All buttons use Mui-button-OFS-button class variants
